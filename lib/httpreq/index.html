HTTPreq.js

HTTPreq.js is a JavaScript library designed to simplify xHTTP requests in JS3 projects.
It provides an easy acces to methods and JavaScript adaptations for adapting it to every requests types.


Using Exemple: HTML <div id="EmbedId"></div> <script>includeHTML("EmbedId", "Path/To/File");</script> To obtanin the content of a file and embed it in any elements with the indiqued ID. (Here, EmbedId.)
