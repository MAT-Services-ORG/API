HTTPreq.js

HTTPreq.js is a JavaScript library designed to simplify xHTTP requests in JS3 projects.
It provides an easy acces to methods and JavaScript adaptations for adapting it to every requests types.

Usage

To use HTTPreq.js, include the library in your project:

<script src="httpreq.js"></script>

Then, you can make requests using the provided methods.
